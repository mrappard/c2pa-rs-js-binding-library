# C2PA Rust JavaScript Library

This is a TypeScript library that leverages Rust via WebAssembly.

## Project Structure

- `crate/`: Rust source code.
- `src/`: TypeScript source code.
- `pkg/`: WebAssembly package generated by `wasm-pack`.
- `dist/`: Compiled TypeScript library.

## Getting Started

### Prerequisites

- [Rust and Cargo](https://rustup.rs/)
- [wasm-pack](https://rustwasm.github.io/wasm-pack/installer/)
- [Node.js and npm](https://nodejs.org/)

### Building

To build both the Rust/WASM module and the TypeScript library, run:

```bash
npm run build
```

This will:
1. Compile the Rust code in `crate/` to WASM in `pkg/`.
2. Compile the TypeScript code in `src/` to JavaScript in `dist/`.

## Usage

```typescript
import { sayHello } from 'c2pa-rs-javascript-library';

console.log(sayHello()); // Output: Hello from Rust!
```

## Scripts

- `npm run build:wasm`: Build only the Rust/WASM module.
- `npm run build:ts`: Build only the TypeScript library.
- `npm run build`: Build everything.
